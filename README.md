# Seekurify

**Seekurify** is an all-in-one cybersecurity platform and secure password manager. It combines an encrypted credential vault with AI-powered security tools, real-time threat detection, incident response automation, and interactive security awareness training.

The application lives in [`main/seekurify-main/`](main/seekurify-main/) — see that folder's [README](main/seekurify-main/README.md) for full setup, environment configuration, API reference, and deployment instructions.

## Repository Layout

| Path | Description |
|------|-------------|
| [`main/seekurify-main/`](main/seekurify-main/) | Full application source — React/TypeScript frontend, Express backend, Chrome extension, tests |
| `LICENSE` | MIT license |

## Quick Start

```bash
cd main/seekurify-main
npm install
npm run dev:full
```

For environment variables, API endpoints, the Chrome extension, and deployment details, see [`main/seekurify-main/README.md`](main/seekurify-main/README.md).
